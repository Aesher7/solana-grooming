🐾 Solana Dog Grooming — Website Project

A high-converting, fully responsive website for Solana Dog Grooming, a local pet grooming business based in Craig, Colorado. Built with Google AI Studio using the Gemini 2.5 Pro model.


📌 Project Overview

This website was designed to attract and convert local customers searching for dog grooming services in the Craig, CO area. It includes an online booking form, services/pricing pages, a photo gallery, and a privacy policy compliant with the Colorado Privacy Act (CPA).
